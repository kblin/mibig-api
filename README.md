MIBiG API
=========

The WebAPI server driving the [Minimum Information about a Biosynthetic Gene
cluster (MIBiG)](http://mibig.secondarymetabolites.org/) repository.


Developing
----------

Make sure you have a recent version of Go (1.22.x).
